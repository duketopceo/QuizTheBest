# Branch Guide

## Quick Reference

- **`main`**: Core application (Backend + Frontend web app)
- **`android-app`**: Android mobile app development

## Getting Started

### Working on Core App (Backend/Frontend)
```bash
git checkout main
```

### Working on Android App
```bash
git checkout android-app
```

## Configuration

Both branches use the same `config.json` file structure. See [docs/CONFIG_SETUP.md](docs/CONFIG_SETUP.md) for setup instructions.
